Discrete Time Function

Be sure to import Symbulate using the following commands.

from symbulate import *
%matplotlib inline

Instead of using RandomProcess, we can use DiscreteTimeFunction in combination with RV in Symbulate for RP's on a discrete set of values.

This function takes the following arguments:

  • fs - the sampling rate of the process. Defaults to 1.
  • index_set - can be set to Naturals() or Reals(). fs overwrites this argument.
  • func - defaults to f[n] = n / fs.

This function creates an infinite list of discrete values to work with. We can then manipulate RV's over these values.

P = BoxModel([0, 1, 2], size=2)
A, B = RV(P)
n = DiscreteTimeFunction(fs=1/3)
X = A + B * n
X.sim(10).plot()
